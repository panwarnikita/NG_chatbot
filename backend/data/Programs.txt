## 💻 School of Programming (SOP): A Complete Overview
### 📝 Introduction - The School of Programming (SOP) at NavGurukul is a practical learning program designed to help students become skilled software developers from scratch.
The curriculum focuses on learning by building real projects, solving problems, and understanding how modern software systems work.
Students start with basic web development and gradually move towards advanced programming, AI integration, backend systems, and real-world projects.
The goal of SOP is to help learners:
Build strong programming fundamentals
Develop problem-solving ability
Learn modern web technologies
Build real applications
Work with AI tools
Become job-ready developers
The entire program follows a "learning by doing" approach, where students continuously practice through projects, mini-projects, and practical assignments.

### 🧠 Learning Philosophy of SOP
The School of Programming is based on a few important learning principles:

Learning by Doing: Students learn best when they build things themselves. Instead of only studying theory, learners build projects from day one.

AI as a Learning Partner: Students are encouraged to use AI tools to support their learning, helping them explore ideas, debug code, and experiment.

Anyone Can Learn Programming: The program is designed for beginners. Even students without prior coding knowledge can learn step-by-step.

Peer Learning: Students learn together by discussing problems, explaining concepts, and reviewing each other's work. Teaching others helps deepen understanding.

Daily Planning & Discipline: Students learn to manage their learning through daily planning and consistent practice.

### 🏗️ Program Structure of School of Programming
The SOP curriculum is divided into multiple learning phases, where each phase focuses on new concepts and technologies. Students gradually move from:
Basic Web Development → JavaScript → Backend → Databases → AI → Full-stack projects.

📅 Program Phases
🔹 Induction Phase
Duration: ~30 days

This phase prepares students for the learning journey.

Students learn: How to learn programming effectively, problem-solving mindset, number systems, flowcharts, logic building, and daily learning habits.

Goal: To build thinking ability before coding.

## 🔹 Phase 1 — HTML Foundations
Project: Student Profile & Course Portal

Students build a multi-page website using only HTML to understand structure.

Pages: Home, Profile, Courses, Feedback, Grades table, and Contact page.

Topics: HTML structure, headings, paragraphs, lists, links, images, tables, forms, and semantic HTML.

Activity: Students record videos explaining their code to demonstrate understanding.

## 🔹 Phase 2 — CSS (Styling Websites)
Students make websites visually attractive by applying CSS to the Phase 1 project.

Topics: CSS syntax, selectors, colors, fonts, spacing (margin, padding), borders, box model, and layout systems.

Modern Tools: Flexbox and CSS Grid.

Skill: Responsive design for mobile and desktop devices.

## 🔹 Phase 3 — JavaScript Fundamentals
Makes websites interactive.

Project: Interactive Quiz Application (Start quiz, dynamic questions, scoring, results, restart).

Topics: Variables, data types, operators, arrays, objects, loops, functions, conditional statements, DOM manipulation, and event handling.

## 🔹 Phase 4 — AI-Powered Content Generator
Students start working with Artificial Intelligence APIs.

They build a web application that generates content using Gemini AI.

Topics: JavaScript ES6 features, API integration, sending requests to AI models, and displaying responses.

## 🔹 Phase 5 — AI Chat Application (Node.js Backend)
Project: "Ask Gemini" Web App

Students learn backend development using Node.js and Express.js.

Topics: Backend server creation, API routes, request/response handling, and connecting frontend with backend.

## 🔹 Phase 6 — Databases
Students learn how to store and manage data using MongoDB and Mongoose.

Project: Student Feedback Manager

Topics: Database concepts, data storage, schema design, CRUD operations, and connecting backend with database.

## 🔹 Phase 7 — React Bootcamp
Students learn React, a modern frontend framework used in industry.

Topics: React components, state management, props, UI building, and frontend architecture.

## 🔹 Final Project
Students build an open-ended project (Real product, social impact solution, or a paid project).

Goal: Apply everything learned during the program.

🛠️ Skills Students Gain
Programming: JavaScript, problem solving, logic building.

Web Development: HTML, CSS, responsive design, React.

Backend Development: Node.js, Express.js, API development.

Databases: MongoDB, data management.

AI Integration: AI APIs, building AI-powered applications.

Software Engineering: Debugging, project building, explaining code, teamwork, and documentation.

## 📖 Learning Methods Used in SOP
Project Based Learning: Real projects in every phase.

Concept Videos: Explaining concepts through recorded videos.

Mini Projects: Reinforcing concepts.

Peer Learning: Collaboration and support.

Assessments: Evaluated using rubrics focusing on understanding, code structure, and communication.

## 🏆 Outcome of the Program
After completion, students will be able to:

Build complete web applications

Write clean JavaScript code

Create responsive websites

Develop backend APIs

Store and manage data

Integrate AI into applications

Build real-world projects

### Certificate Program in Accounting, Costing, Taxation & Digital Accounting Tools
## 📋 1. PROGRAM OVERVIEW
Program Duration: 13–15 Months (Approx. 12 months of structured learning + revision & assessments)

Eligibility: 12th Pass (Any Stream)

Pedagogical Flow: Concept → Task → Games → Case → Reflection

## 🎯 Core Objectives
Develop strong accounting and financial interpretation skills

Build advanced Excel and digital accounting tool proficiency

Enable students to read, analyze, and interpret financial statements

Develop communication, analytical, and self-learning skills

Encourage AI-assisted learning, reading of amendments, case laws, and financial news

## 🏫 Teaching Methodology
Peer-to-peer learning & Mentor–mentee learning

Case studies (corporate, NGO, MSME) & Group assignments

Simulations and games

Tool-based practical learning (Excel, Tally, Zoho Books)

## 🏆 Program Learning Outcome
By the end of the program, students will be able to:

Read and interpret financial statements of companies, firms, and non-profits

Apply accounting concepts in real-life business scenarios

Prepare and analyze financial, cost, and management reports

Use Tally and digital tools confidently

Understand tax implications for individuals, firms, and companies

## 📖 Module 1: Foundations of Financial Accounting
Objective: Build strong conceptual clarity of accounting principles and recording mechanisms.

1.1 Introduction to Accounting: Meaning, scope, objectives, advantages & limitations; Book-keeping vs accounting; Users of accounting information.

1.2 Accounting Concepts & Principles: Business entity, money measurement, going concern; Accrual, prudence, consistency, cost concept.

1.3 Accounting Terminology & Accounting Equation: Assets, liabilities, capital, drawings; Accounting equation and its applications.

1.4 Double Entry System & Journal: Golden rules of accounting; Cash and credit transactions; Journal entries from day-to-day business activities.

## 📖 Module 2: Recording, Classification & Control of Accounts
Objective: Enable students to record transactions systematically and ensure accounting accuracy.

2.1 Ledger & Balancing: Posting journal entries to ledger; Balancing of ledger accounts.

2.2 Subsidiary Books: Cash book, purchase book, sales book; Practical classification through activities and games.

2.3 Trial Balance: Meaning, objectives, preparation, and limitations.

2.4 Errors & Rectification: Types of accounting errors; Rectification of errors and suspense account.

2.5 Bank Reconciliation Statement (BRS): Cash book vs pass book; Preparation of BRS and common reconciliation items.

## 📖 Module 3: Final Accounts & Business Accounting
Objective: Develop ability to prepare and interpret financial statements of different business forms.

3.1 Final Accounts – Sole Proprietor: Trading account; Profit & loss account with adjustments; Balance sheet preparation.

3.2 Partnership Accounting: Partnership deed and profit-sharing ratio; Admission, retirement, goodwill, and revaluation.

3.3 Company Accounts: Types of companies; Share capital and basic share issue concepts; Preparation of company financial statements; Introduction to cash flow statement.

## 📖 Module 4: Management & Cost Accounting
Objective: Strengthen managerial decision-making and cost control skills.

4.1 Management Accounting Fundamentals: Nature, scope, objectives; Financial accounting vs management accounting.

4.2 Financial Statement Analysis: Comparative statements; Common size statements; Trend analysis.

4.3 Ratio Analysis: Liquidity, profitability, solvency, and activity ratios; Interpretation using real company data.

4.4 Working Capital & Capital Budgeting: Operating cycle and working capital management; Capital budgeting techniques (Payback, NPV, IRR).

4.5 Cost Accounting Basics: Classification and elements of cost; Material costing and inventory control; Labour costing and overheads; Introduction to standard costing and variance analysis.

## 📖 Module 5: Taxation – Income Tax & GST
Objective: Enable learners to understand tax laws, compute taxes, and ensure compliance.

5.1 Introduction to Income Tax: Assessee, assessment year, previous year; Residential status and scope of total income.

5.2 Heads of Income: Salary, House property, PGBP (Profits and gains of business or profession), Capital gains, and Income from other sources.

5.3 Deductions & Computation: Deductions under Chapter VI-A (80C, 80D, 80E, etc.); Step-by-step computation of total income.

5.4 Introduction to GST: Concept and objectives; Indirect taxes before GST overview; CGST, SGST, IGST meaning and applicability.

5.5 GST Compliance: Registration process and thresholds; Input Tax Credit (ITC) concept; Overview of GST returns (GSTR-1, GSTR-3B); Basic GST compliance from an accounting perspective.

## 📖 Module 6: Digital Accounting Tools – Tally Prime
Objective: Provide hands-on experience in computerized and GST-enabled accounting.

6.1 Introduction: Installation, interface, company creation, and basic configuration.

6.2 Masters Setup: Creation of ledgers and groups; GST-related ledger configuration.

6.3 Accounting Vouchers: Payment, receipt, contra, and journal vouchers; Recording GST transactions.

6.4 Reports & Reconciliation: Day book, trial balance, P&L account, balance sheet, and bank reconciliation in Tally; GST reports overview.

🏁 Overall Program Outcomes
Maintain manual and computerized books of accounts.

Prepare and analyze financial statements.

Apply cost and management accounting tools for decision-making.

Compute income tax and understand GST compliance.

Use Tally Prime confidently for accounting and GST-enabled transactions.

## SOSC (School of second chance) Future Textile Designer Programme
Creating Climate-Responsive Livelihoods for Women Survivors
1. Sneak Peek into the Future Textile Designer Programme
The Future Textile Designer Programme is a residential, future-skills livelihood initiative tailored for women from highly vulnerable backgrounds—survivors of violence, trafficking, wrongful incarceration, and women living with HIV.
The programme moves beyond traditional tailoring by equipping participants with advanced textile, design, digital, and business skills that open pathways to dignified and sustainable employment.
Programme Outcomes
Each graduate should be able to:
Secure stable employment earning at least ₹20,000 per month
Form a self-help group
Establish an independent design-led enterprise

### 2. Why School of Second chance  Programme?
Conventional tailoring training often leads to low-wage and unpredictable jobs.
At the same time, the fashion industry is rapidly shifting toward:
Sustainability
Upcycling
Responsible production
Digital commerce
These fields require new technical and creative capabilities.
This programme addresses this gap by preparing women for roles in the emerging circular economy, while building long-term financial resilience and confidence.

### 3. Programme Structure of sosc
Duration: 12–18 months (self-paced)
Format: Residential
The programme includes:
Food and accommodation
Digital devices
English language support
Psychosocial care
Peer learning environment
Internship with ethical brands or enterprises

### 4. Key Curriculum Components of SOSC
A. Technical & Product Development Skills
Participants learn to move beyond basic tailoring and conceptualize original products.
Skills Covered
Advanced sewing and finishing
Commercial-level product construction
Pattern-making
Sampling and prototyping
Quality control and industry standards
These skills build a strong technical foundation suitable for design studios and sustainable fashion enterprises.

B. Design Thinking & Creative Development
Participants develop creative and design capabilities through:
Fundamentals of product and textile design
Trend research
Material exploration
Creating market-ready accessories, garments, and home products

C. Upcycling & Circular Textile Innovation (Programme’s Core Innovation)
This module focuses on sustainability and responsible production.
Key Learning Areas
Understanding textile waste streams
Techniques to transform post-consumer and post-production waste
Designing functional and desirable upcycled products
Waste mapping and responsible sourcing
This prepares women for the fashion industry’s transition toward circularity, waste reduction, and climate-aligned manufacturing, helping them stand apart from their contemporaries.

D. Business, Financial & Entrepreneurial Skills
Participants gain practical business skills such as:
Costing and pricing
Negotiation and order management
Budgeting
Micro-enterprise planning
Understanding market cycles and customer needs
These skills enable women to:
Secure fair job opportunities
Launch their own ventures
Lead negotiations confidently to ensure fair and equitable pay

E. Digital Skills & Communication
Participants build digital literacy and communication skills including:
Digital marketing and online sales platforms
Social media storytelling
Basic product photography
Portfolio creation
English language and communication skills
Digital fluency helps graduates:
Reach wider markets
Become future-job ready
Build economic resilience

5. Climate Lens & Adaptive Livelihood Strategy
The programme integrates a strong climate-resilience approach.
